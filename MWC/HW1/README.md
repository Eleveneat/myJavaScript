#在给定源代码：完成@+环形菜单的CSS。
***

1. @+环形菜单出现在页面下部，靠近底边，居中
其关闭时（鼠标未指向），只显示缩小的@+按钮
2. 鼠标指向之后，显示出环形菜单，包括5个控制按钮和一个信息栏（info-bar） 
3. 颜色：按钮蓝色：rgba(48, 63, 159, 1)；绿色字体：#079E6E；各种灰色，请自行用rgba来调整。
注意：本次作业没有任何JavaScript，动画部分可用CSS3的transition来实现。

- CSS transition
- 嵌套CSS规则
